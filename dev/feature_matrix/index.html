Direct matrix/tensor comparison, PCC/Euclidean motif metrics, the motif CLI command, and the "motif" null strategy are deliberately removed.

GPU/distributed execution, ZIP bundles, automatic scientific-workflow cache integration, and empirical GEV fallback are not current public contracts.
